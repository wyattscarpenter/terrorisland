Lewis’s Notes

This strip reminds me of a philosophical paradox. We can ask what the most useful question/answer pair is. And, presumably, the answer to that question would be really useful to know. Perhaps, the most useful thing to know.

So then, that question and its answer would be the most useful question/answer pair. So the answer to that question would be that question and its answer.

This is not the cleanest paradox, as there are a number of different ways you might respond, and the set up is fairly confusing. Zeno’s paradoxes and the Liar paradox are way nicer, but this one is still interesting.
