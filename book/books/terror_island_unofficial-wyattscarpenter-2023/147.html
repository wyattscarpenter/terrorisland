Ben’s Notes

Here’s something cool I found out about on Wikipedia. Goodstein’s theorem, which is that every Goodstein sequence converges to 0, has a pretty simple proof using infinite ordinals, but it can be shown to be unprovable within Peano arithmetic. It’s a really natural-looking theorem as examples of undecidability go, and the proof using ordinals is easy to follow. The proof that it’s unprovable in PA is trickier, unfortunately.
